# Gradient Index Lenses

[![PDF](https://img.shields.io/badge/View-PDF-red)](/main.pdf)

This repository contains the LaTeX source code and compiled PDF for the paper:  
**"Gradient Index Lenses"**  
By **Eduardo Barroso Portugal**

## Notes
- This is a pre-print version.
- `document_layout.cls` is a custom class for the document layout.
- The bibliography is generated using BibLaTeX. Its source is in the file `references.bib` and its style is defined in the file `document_layout.cls`.
